
База данных для курсового проекта создана на основе web-ресурса www.receptmira.ru для СУБД MySQL.

Предназначена для хранения, поиска, ввода новых рецептов разных стран мира пользователями и общения пользователей.

База данных позволяет решать следующие задачи:
1. заполнение пользователем личной страницы;
2. добавление пользователем новых продуктов, рецептов, фотографий, друзей;
3. при добавлении/удалении контента (фото, друзья, лайки, сообщения, рецепты) 
   на странице пользователя триггерами увеличиваются/уменьшаются одноименные счетчики;
4. обмен личными сообщениями;
5. ставить лайки на контент пользователей;
6. формирование ленты новостей при вводе пользователем нового контента;
7. выбор рецепта по характеристикам: континент, страна, категория блюда, автор, время приготовления, 
   пищевая ценность, дата создания;
8. автоматическое формирование облака часто используемых основных продуктов рецептов;
9. составление меню с отбором рецептов в таблицу menu;
10. составление листа покупок на основании таблицы menu и установленного количества порций. 

База данных состоит из 14 таблиц, 3 представлений, 2 процедур, 10 триггеров.
